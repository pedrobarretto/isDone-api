## IsDone - Your todo app

IsDone is a to-do API for you to organize your daily tasks.
Front end on the way.

### Stack of API:
- Node.js
- Typescript
- Express
- JWT
- MongoDB
- Mongoose
- bcryptjs

### Features
- [X] Complete CRUD of to-dos and users
- [X] Authentication
- [X] Docker
- [ ] Swagger for documentation
